My Projects

Here are some of the projects I've worked on

Hospital Webpage using Angular

Users search doctor availability, schedule appointments, and manage login/signup.

Multiple Disease Prediction using Machine Learning

Built a static web app using Streamlit to predict Hepatitis C and Prostate Cancer using Machine Learning algorithms. Optimized for best prediction accuracy.

Crop Management System using Machine Learning

Developed a web application that predicts crop yield and recommends suitable crops based on weather data and weather forecasts using Machine Learning algorithms.

AI-Powered X-Ray Diagnosis Using CNN & RNN

Developed a Gradio web application which combines Convolutional Neural Networks (CNNs) for feature extraction and Recurrent Neural Networks (RNNs) for text generation, enabling automated X-ray disease diagnosis. It employs Grad-CAM for visual interpretability and Gradio for an interactive web-based interface.

Optimizing-Traffic-Light-Control-using-Q-Learning

Developed an intelligent traffic signal control system using Q-Learning, a reinforcement learning algorithm, to optimize traffic flow at intersections. The model was trained using a SUMO Traffic Simulator and TensorFlow/Keras, enabling adaptive signal timing based on real-time traffic conditions. This approach reduces congestion, improves vehicle throughput, and minimizes waiting time.

Resume Categorizer

Developed an NLP-based tool on Streamlit to analyze and classify resumes by skills and job roles. Created an Interface for easy visualization and filtering of categorized resumes.
